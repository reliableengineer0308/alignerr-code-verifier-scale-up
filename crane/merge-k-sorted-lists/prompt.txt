Problem: Merge K Sorted Lists

## Problem Description
You are given an array of K sorted linked lists. Each linked list is sorted in ascending order. 
Merge all the linked lists into one sorted linked list and return its head.

## Input Format
- First line: T (number of test cases)
- For each test case:
  - First line: K (number of linked lists)
  - Next K lines: each line contains space-separated integers representing one sorted linked list

## Output Format
- For each test case: space-separated integers of the merged sorted linked list

## Constraints
- 0 ≤ K ≤ 10^4
- 0 ≤ total number of nodes ≤ 10^4
- -10^4 ≤ node value ≤ 10^4

## Example 1: Basic Case
### Input
1
3
1 4 5
1 3 4
2 6

### Output
1 1 2 3 4 4 5 6

### Explanation
Three sorted lists:
- List 1: 1 → 4 → 5
- List 2: 1 → 3 → 4  
- List 3: 2 → 6

Merged: 1 → 1 → 2 → 3 → 4 → 4 → 5 → 6

## Example 2: Empty Lists
### Input
1
0

### Output
(empty line)

### Explanation
No lists to merge, output is empty.

## Example 3: One List
### Input
1
1
1 2 3 4 5

### Output
1 2 3 4 5

Only one list, output is the same as input.

## Algorithm Requirements
- Use min-heap for efficient merging
- Time complexity: O(N log K) where N is total nodes
- Space complexity: O(K)
