Problem: Maximum Subarray Sum with Constraints

Description:
Implement a function max_subarray_sum_constrained(arr, min_len, max_len) that finds the maximum sum of a contiguous subarray with length constraints.

Your task:
- Find the contiguous subarray of length L where min_len ≤ L ≤ max_len
- Maximize the sum of elements in this subarray
- Return the maximum possible sum
- Handle edge cases: negative numbers, empty results, large arrays

Input:
- Integer array arr (can be empty)
- min_len: minimum subarray length (≥ 1)
- max_len: maximum subarray length (≥ min_len)
- Array size: 0 ≤ n ≤ 10⁵
- Element range: −10^4≤arri≤10^4
 
Output:
- Integer: maximum sum of any contiguous subarray with min_len ≤ length ≤ max_len
- If no valid subarray exists, return 0

Examples:
Example 1:
Input: arr = [1, -2, 3, 4, -1, 2], min_len = 2, max_len = 4
Output: 8
Explanation: Subarray [3, 4, -1, 2] has length 4 and sum = 8 (maximum possible)

Example 2:
Input: arr = [-1, -2, -3], min_len = 1, max_len = 3
Output: -1
Explanation: Best is single element [-1] (sum = -1)

Example 3:
Input: arr = [5, 2, -1, 3], min_len = 3, max_len = 3
Output: 6
Explanation: Only subarrays of length 3: [5,2,-1] → sum=6, [2,-1,3] → sum=4 → max=6

Constraints:
0 ≤ n ≤ 10^5
1≤min_len≤max_len≤n (if n>0)
Time complexity: O(n) or better
Space complexity: O(1) extra space
Avoid brute-force O(n^2) solutions

Consider both positive and negative numbers

Notes:
- Use sliding window or optimized DP approach
- Precompute prefix sums if needed
- Handle cases where all numbers are negative
- Edge case: empty array → return 0
- The subarray must be contiguous
- You may assume min_len ≤ max_len always holds
