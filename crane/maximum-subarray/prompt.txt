Problem: Maximum Subarray

Description:
Implement a function max_subarray(nums) that finds the contiguous subarray (containing at least one number) which has the largest sum and returns its sum.

Your task:
- Find the maximum sum of any contiguous subarray
- Use Kadane’s algorithm for O(n) time complexity
- Handle edge cases (all negative, single element, etc.)

Input:
- List nums of integers (1 ≤ len(nums) ≤ 10⁵)
- Each num satisfies -10⁴ ≤ num ≤ 10⁴

Output:
Integer: maximum sum of contiguous subarray

Examples:
Example 1:
Input: nums = [-2,1,-3,4,-1,2,1,-5,4]
Output: 6
Explanation: [4,-1,2,1] has largest sum = 6

Example 2:
Input: nums = [1]
Output: 1

Example 3:
Input: nums = [5,4,-1,7,8]
Output: 23

Constraints:
1 ≤ len(nums) ≤ 100000
-10000 ≤ nums[i] ≤ 10000
Time complexity: O(n)

Space complexity: O(1)

Notes:
- Use Kadane’s algorithm:
- Track current subarray sum (current_sum)
- Track global maximum (max_sum)
- At each step: current_sum = max(num, current_sum + num)
- Update max_sum if current_sum is larger
- Handle all-negative arrays (result will be the largest single element)
