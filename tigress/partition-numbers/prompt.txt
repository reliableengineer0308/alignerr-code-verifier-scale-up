Problem: Partition Numbers

Description:
The partition number p(n) represents the number of distinct ways of writing n as a sum of positive integers (order doesn't matter).

For example:
p(4) = 5 because:
4 = 4
4 = 3 + 1
4 = 2 + 2
4 = 2 + 1 + 1
4 = 1 + 1 + 1 + 1

The partition numbers can be computed using the recurrence relation:
p(n) = Σ[k≠0] (-1)^(k+1) * [p(n - k(3k-1)/2) + p(n - k(3k+1)/2)]
with p(0) = 1

Given n and a modulus M, compute p(n) modulo M.

Input:
- First line: T (number of test cases, 1 ≤ T ≤ 100)
- For each test case:
  - n M (0 ≤ n ≤ 1000, 1 ≤ M ≤ 10^9)

Output:
- For each test case:
  - Print p(n) % M

Examples:

Example 1:
Input:
6
0 1000000007
1 1000000007
2 1000000007
3 1000000007
4 1000000007
5 1000000007

Output:
1
1
2
3
5
7

Explanation:
Case 1: p(0) = 1 (empty partition)
Case 2: p(1) = 1
Case 3: p(2) = 2 (2, 1+1)
Case 4: p(3) = 3 (3, 2+1, 1+1+1)
Case 5: p(4) = 5
Case 6: p(5) = 7

Constraints:
- 1 ≤ T ≤ 100
- 0 ≤ n ≤ 1000
- 1 ≤ M ≤ 10^9
