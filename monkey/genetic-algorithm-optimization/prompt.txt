Problem: Genetic Algorithm for Multimodal Function Optimization

Description:
Find the global maximum of a multimodal function using Genetic Algorithm.
The function is: f(x) = sin(x) + sin(10*x/3) + log(x) - 0.84*x + 3, for x in [2.7, 7.5]

Input:
- First line: population_size generations mutation_rate
- No additional input for function definition

Output:
- First line: best_x best_fitness (both rounded to 6 decimal places)
- Second line: convergence_data (comma-separated best fitness per generation)

Constraints:
- 50 ≤ population_size ≤ 1000
- 100 ≤ generations ≤ 5000
- 0.01 ≤ mutation_rate ≤ 0.2

Examples:

Example 1:
Input:
100 1000 0.05

Output:
5.146423 3.203145
0.512,0.893,1.234,1.567,...,3.203

Example 2:
Input:
200 2000 0.1

Output:
5.146423 3.203145
0.623,1.015,1.478,1.892,...,3.203

Implementation Approach:
1. Use real-valued encoding for chromosomes
2. Tournament selection
3. Blend crossover (BLX-α)
4. Gaussian mutation
5. Elitism to preserve best solution
