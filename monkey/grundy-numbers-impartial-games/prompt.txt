Problem: Grundy Numbers for Composite Impartial Games

Description:
Two players play a composite impartial game consisting of multiple independent subgames.
Each subgame has its own rules for moves. On each turn, a player chooses one subgame and makes a valid move in that subgame.
The player who cannot make any move loses.

For each subgame type, calculate its Grundy number:
1. Take-away game: Start with n stones. On each move, remove 1, 2, or 3 stones.
2. Power game: Start with n. On each move, remove a perfect square number of stones (1, 4, 9, 16, ...)
3. Division game: Start with n. On each move, replace n with ⌊n/2⌋ or ⌊n/3⌋

Given the composite game configuration, determine if the first player has a winning strategy.

Input:
- First line: T (number of test cases)
- For each test case:
  - First line: G (number of subgames)
  - Next G lines: Each line contains the game type and initial value:
    - "take n" for take-away game
    - "power n" for power game  
    - "divide n" for division game

Output:
- For each test case: "First" if first player wins, "Second" otherwise

Constraints:
- 1 ≤ T ≤ 1000
- 1 ≤ G ≤ 100
- 1 ≤ n ≤ 10^6

Examples:

Output:

Explanation:

- take(5): Grundy number = 5 % 4 = 1
- power(8): Calculate Grundy for power game

XOR of all Grundy numbers ≠ 0 → First wins
