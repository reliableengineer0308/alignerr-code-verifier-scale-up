Problem: Ant Colony Optimization for Shortest Path

Description:
Find the shortest path from start to end in a grid with obstacles using Ant Colony Optimization.
Each ant probabilistically moves based on pheromone trails and heuristic information.

Input:
- First line: N M (grid size: rows, columns)
- Next N lines: M characters ('.' for empty, '#' for obstacle, 'S' for start, 'E' for end)
- Next line: num_ants num_iterations evaporation_rate

Output:
- First line: path_length
- Next line: path (comma-separated coordinates in format "row,col")

Constraints:
- 10 ≤ N, M ≤ 100
- 10 ≤ num_ants ≤ 100
- 10 ≤ num_iterations ≤ 1000
- 0.1 ≤ evaporation_rate ≤ 0.5

Examples:

Example 1:
Input:
10 10
..........  
.#.#.#.#..
.#.#.#.#..
.S........
.#.#.#.#..
.#.#.#.#..
.......#..
.#.#.#.#..
.#.#.#.E..
..........
50 100 0.3

Output:
14
3,0 3,1 3,2 3,3 3,4 3,5 3,6 3,7 3,8 4,8 5,8 6,8 7,8 8,8

Example 2:
Input:
8 8
........
.#.#.#..
.#.#.#..
.S...#..
.#.#.#..
.#.#.#..
......E.
........
30 50 0.2

Output:
10
3,1 3,2 3,3 3,4 3,5 4,5 5,5 6,5 6,6 6,7

Implementation Approach:
1. Use Ant System algorithm with pheromone trails
2. Ants probabilistically choose next move based on pheromone and heuristic
3. Update pheromones after each iteration with evaporation
4. Use Manhattan distance as heuristic to guide ants toward target
5. Return the best path found across all iterations
